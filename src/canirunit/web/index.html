Quantization

Weight quant is intrinsic on MLX and Ollama (set by the repo/tag). KV quant is a separate lever — dropping it to q4 often unlocks the full native context.

Memory vs. context

Weights are constant; KV cache grows linearly with context. The comfort line is what fits without spilling; the hard line (Apple only) is what still loads via compression/swap, slower.

Decode speed vs. context

Decode is memory-bandwidth-bound. As KV grows the per-token bytes read grow with it, so speed drops. Dashed = estimated; solid = measured (after calibration).
